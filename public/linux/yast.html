YaST

Sieć i połączenia

  • Konfiguracja adresacji IP, maski, bramy domyślnej i DNS:
    System → Network Settings (Ustawienia sieciowe)
  • Zmiana nazwy komputera (Hostname):
    System → Network Settings → zakładka Hostname/DNS
  • Konfiguracja zapory sieciowej (otwieranie portów):
    Security and Users → Firewall (Zapora sieciowa)

Zarządzanie systemem i kontami

  • Tworzenie użytkowników, przypisywanie haseł i grup:
    Security and Users → User and Group Management (Zarządzanie użytkownikami i grupami)
  • Instalacja i usuwanie oprogramowania (np. serwera Apache):
    Software → Software Management (Zarządzanie oprogramowaniem)
  • Włączanie i wyłączanie usług systemowych (zastępuje systemctl):
    System → Services Manager (Menedżer usług)

Dyski i zasoby

  • Partycjonowanie dysków, formatowanie i montowanie:
    System → Partitioner (Partycjoner)
  • Konfiguracja udostępniania plików w sieci (Samba):
    Network Services → Samba Server (wymaga wcześniejszego zainstalowania pakietów)
