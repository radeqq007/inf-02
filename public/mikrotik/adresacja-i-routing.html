Adresacja IP i Routing w MikroTik (RouterOS / WinBox)

Teoria

  • Adres IP musi być przypisany do konkretnego interfejsu fizycznego (np. ether1) lub logicznego (np. bridge)
  • Trasa dynamiczna (DAC) – Dynamic Active Connected; tworzona automatycznie przez system po przypisaniu adresu IP do interfejsu
  • Brama domyślna (Default Gateway) – trasa statyczna dla ruchu docelowego do sieci zewnętrznych (oznaczana jako 0.0.0.0/0)
  • Dystans (Distance) – określa priorytet trasy w tabeli routingu; trasa z niższym dystansem jest wybierana w pierwszej kolejności
  • Aktywność trasy (Active) – trasa jest aktywna (oznaczenie A), gdy jej brama (gateway) jest osiągalna (reachable)

Konfiguracja przez WinBox (GUI)

  • Dodawanie adresu IP: Menu główne → IP → Addresses → Kliknij niebieski plus + → W polu Address wpisz np. 192.168.88.1/24 → W polu Interface wybierz odpowiedni interfejs (np. bridge) → Kliknij Apply i OK
  • Dodawanie trasy domyślnej (Routing): Menu główne → IP → Routes → Zakładka Routes → Kliknij niebieski plus + → W polu Dst. Address zostaw 0.0.0.0/0 → W polu Gateway wpisz adres IP bramy dostawcy (np. 192.168.1.1) → W polu Distance ustaw priorytet (np. 1) → Kliknij OK
  • Podgląd i usuwanie: Listy w oknach Address List oraz Route List pozwalają na przeglądanie wpisów. Kliknięcie czerwonego minusa - usuwa zaznaczony rekord.

Konfiguracja przez CLI (Komendy)

  • /ip address add address=192.168.88.1/24 interface=bridge comment="LAN" – przypisuje adres IP z maską do interfejsu bridge
  • /ip address print – wyświetla listę wszystkich skonfigurowanych adresów IP wraz z interfejsami
  • /ip route add dst-address=0.0.0.0/0 gateway=192.168.1.1 distance=1 comment="Brama domyslna" – dodaje domyślną trasę statyczną do internetu
  • /ip route print – wyświetla pełną tabelę routingu (w tym statusy tras: AS, DAC itp.)
