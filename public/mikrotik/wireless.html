Teoria

  • Security Profile (Profil bezpieczeństwa) – zestaw ustawień definiujący standardy szyfrowania (np. WPA2, WPA3) oraz klucz współdzielony (hasło) do sieci bezprzewodowej
  • SSID (Service Set Identifier) – unikalna nazwa rozgłaszanej sieci Wi-Fi, widoczna dla urządzeń klienckich
  • Tryb ap-bridge (Access Point Bridge) – podstawowy tryb pracy karty radiowej, umożliwiający działanie jako punkt dostępowy obsługujący wielu klientów
  • Pasmo (Band) – definiuje standardy i częstotliwość (np. 2ghz-b/g/n lub 5ghz-a/n/ac) optymalizujące prędkiem i zasięg połączenia
  • Tabela rejestracji (Registration Table) – lista zawierająca szczegółowe statystyki aktualnie połączonych klientów (poziom sygnału Tx/Rx, uptime, CCQ)

Konfiguracja przez WinBox (GUI)

  • Tworzenie hasła (Security Profile): Menu główne → Wireless → Zakładka Security Profiles → Kliknij niebieski plus + → W polu Name wpisz nazwę profilu (np. wifi_haslo) → Ustaw Mode=dynamic keys → Zaznacz Authentication Types=WPA2 PSK → W polu WPA2 Pre-Shared Key wpisz swoje hasło do Wi-Fi → Kliknij OK
  • Konfiguracja i włączenie sieci (SSID): Menu główne → Wireless → Zakładka WiFi Interfaces → Kliknij dwa razy na interfejs radiowy (np. wlan1) → Przejdź do zakładki Wireless → Ustaw Mode=ap-bridge → W polu Band wybierz odpowiednie pasmo (np. 2ghz-b/g/n) → W polu SSID wpisz nazwę swojej sieci bezprzewodowej → W polu Security Profile wybierz wcześniej utworzony profil (wifi_haslo) → Kliknij Apply, a następnie kliknij przycisk Enable (niebieski ptaszek) w oknie głównym Interface List, aby uruchomić nadawanie
  • Podgląd podłączonych użytkowników: Menu główne → Wireless → Zakładka Registration. Znajdziesz tu listę MAC adresów podłączonych klientów wraz z siłą ich sygnału.

Konfiguracja przez CLI (Komendy)

  • /interface wireless security-profiles add name=wifi_profile mode=dynamic-keys authentication-types=wpa2-psk unicast-ciphers=aes-ccm group-ciphers=aes-ccm wpa2-pre-shared-key="SuperTajneHaslo" – tworzy bezpieczny profil szyfrowania WPA2-PSK
  • /interface wireless set [find default-name=wlan1] mode=ap-bridge ssid="MojeWiFi" band=2ghz-b/g/n frequency=auto security-profile=wifi_profile disabled=no – konfiguruje i włącza interfejs bezprzewodowy wlan1
  • /interface wireless registration-table print – wyświetla listę podłączonych urządzeń bezprzewodowych wraz z parametrami sygnału
