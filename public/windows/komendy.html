Windows Server

Zarządzanie użytkownikami i grupami

  • net user nazwa hasło /add – dodaje nowego użytkownika lokalnego
  • net user nazwa /delete – usuwa użytkownika
  • net localgroup Administratorzy nazwa /add – dodaje użytkownika do grupy
  • net localgroup nazwa_grupy /add – tworzy nową grupę lokalną
  • net user nazwa /passwordreq:yes – wymusza posiadanie hasła
  • net user nazwa /expires:DD/MM/YYYY – ustawia datę wygaśnięcia konta

Konfiguracja i diagnostyka sieci

  • ipconfig /all – szczegółowe informacje o wszystkich kartach sieciowych
  • ipconfig /flushdns – czyści pamięć podręczną DNS
  • ping -n 4 adres_IP – wysyła 4 pakiety ICMP do podanego adresu
  • tracert adres_IP – pokazuje trasę pakietów do celu
  • netstat -ano – otwarte porty i skojarzone z nimi procesy (PID)
  • nslookup nazwa_domeny – zapytanie DNS
  • netsh interface ip set address "Ethernet" static IP maska brama – ustawia statyczny adres IP
  • netsh interface ip set dns "Ethernet" static adres_DNS – ustawia serwer DNS

Zarządzanie rolami i funkcjami (Server Manager / PowerShell)

  • Install-WindowsFeature -Name nazwa -IncludeManagementTools – instaluje rolę lub funkcję (np. Web-Server dla IIS, AD-Domain-Services)
  • Uninstall-WindowsFeature -Name nazwa – usuwa rolę lub funkcję
  • Get-WindowsFeature – lista wszystkich dostępnych ról i ich stanu
  • Install-ADDSForest -DomainName "domena.local" – promuje serwer do roli kontrolera domeny (Active Directory)

Zarządzanie usługami

  • sc start nazwa_usługi – uruchamia usługę
  • sc stop nazwa_usługi – zatrzymuje usługę
  • sc query nazwa_usługi – sprawdza stan usługi
  • sc config nazwa_usługi start= auto – ustawia automatyczny start usługi
  • Get-Service nazwa – sprawdza usługę przez PowerShell
  • Restart-Service nazwa – restartuje usługę przez PowerShell

Operacje na plikach i katalogach (cmd)

  • dir /a – listuje zawartość katalogu wraz z plikami ukrytymi
  • cd ścieżka – przejście do wskazanego folderu
  • mkdir nazwa – tworzy nowy katalog
  • copy źródło cel – kopiuje plik
  • xcopy źródło cel /E /I – kopiuje katalog razem z podkatalogami
  • del plik – usuwa plik
  • rmdir /s /q katalog – usuwa katalog wraz z zawartością (bez pytania)
  • icacls plik_lub_katalog /grant użytkownik:(F) – nadaje pełne uprawnienia do zasobu

Active Directory (PowerShell)

  • New-ADUser -Name "Jan Kowalski" -SamAccountName "jkowalski" -AccountPassword (Read-Host -AsSecureString) -Enabled $true – tworzy użytkownika AD
  • Remove-ADUser -Identity "jkowalski" – usuwa użytkownika z AD
  • New-ADGroup -Name "nazwa" -GroupScope Global – tworzy grupę w AD
  • Add-ADGroupMember -Identity "nazwa_grupy" -Members "jkowalski" – dodaje użytkownika do grupy AD
  • Get-ADUser -Filter * | Select Name,SamAccountName – lista wszystkich użytkowników AD
  • New-ADOrganizationalUnit -Name "nazwa_OU" -Path "DC=domena,DC=local" – tworzy jednostkę organizacyjną (OU)

Udostępnianie zasobów (Samba / SMB)

  • New-SmbShare -Name "nazwa_udziału" -Path "C:\ścieżka" -FullAccess "Użytkownicy" – tworzy udział sieciowy SMB
  • Get-SmbShare – lista wszystkich udziałów sieciowych
  • Remove-SmbShare -Name "nazwa_udziału" – usuwa udział
  • net share nazwa_udziału=C:\ścieżka /grant:użytkownik,FULL – alternatywa przez cmd

Zapora systemu Windows (Windows Firewall)

  • netsh advfirewall firewall add rule name="nazwa" protocol=TCP dir=in localport=80 action=allow – otwiera port TCP 80
  • New-NetFirewallRule -DisplayName "nazwa" -Direction Inbound -Protocol TCP -LocalPort 443 -Action Allow – to samo przez PowerShell
  • Get-NetFirewallRule | Where-Object {$_.Enabled -eq "True"} – lista aktywnych reguł zapory
  • Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled False – wyłącza zaporę (ostrożnie!)

Diagnostyka systemu

  • eventvwr – otwiera Podgląd zdarzeń (logi systemowe, błędy, ostrzeżenia)
  • tasklist – lista uruchomionych procesów
  • taskkill /PID numer /F – wymusza zakończenie procesu po PID
  • systeminfo – szczegółowe informacje o systemie i sprzęcie
  • wmic logicaldisk get size,freespace,caption – zajętość dysków logicznych
  • sfc /scannow – skanowanie i naprawa uszkodzonych plików systemowych
  • gpupdate /force – wymusza ponowne zastosowanie zasad grupy (GPO)
