Adresacja IP i Serwer DHCP w TP-Link (Web GUI / CLI)

Teoria

  • Adres IP zarządzania przełącznika przypisywany jest domyślnie do interfejsu logicznego VLAN 1.
  • Brama domyślna (Default Gateway) – adres IP routera, do którego przełącznik lub hosty wysyłają ruch poza lokalną podsieć.
  • Serwer DHCP (DHCP Server) – usługa automatycznie przydzielająca adresy IP, maski, bramy oraz serwery DNS urządzeniom w sieci.
  • Pula adresów (Address Pool) – określony zakres adresów IP (od-do), z którego serwer DHCP może czerpać konfigurację dla klientów.
  • Statyczna rezerwacja (DHCP Reservation) – przypisanie konkretnego adresu IP do adresu MAC karty sieciowej danego urządzenia (np. serwera lub drukarki).

Konfiguracja przez Web GUI

  • Zmiana adresu IP LAN urządzenia: Menu główne → Network → LAN → W polu IP Address wpisz np. 192.168.0.2 → W polu Subnet Mask ustaw odpowiednią maskę (np. 255.255.255.0) → Kliknij Save
  • Uruchomienie serwera DHCP: Menu główne → Network → DHCP Server → Zakładka DHCP Server → Zaznacz Enable → W polach Start IP Address i End IP Address wpisz zakres (np. 192.168.0.10 do 192.168.0.100) → W polu Default Gateway wpisz adres IP bramy sieci → Kliknij Save
  • Rezerwacja adresu DHCP: Menu główne → Network → DHCP Server → Zakładka Address Reservation → Kliknij Add New → Wpisz MAC Address oraz przypisany mu Reserved IP Address → Ustaw status na Enabled → Kliknij Save

Konfiguracja przez CLI (Komendy)

  • configure – wejście w tryb konfiguracji globalnej systemu
  • interface vlan 1 – wejście do konfiguracji interfejsu logicznego VLAN 1
  • ip address 192.168.0.2 255.255.255.0 – przypisanie adresu IP oraz maski podsieci do interfejsu
  • ip dhcp server – globalne włączenie usługi serwera DHCP na urządzeniu
  • ip dhcp pool LAN_POOL – utworzenie puli DHCP o nazwie "LAN_POOL" i przejście do jej konfiguracji
  • network 192.168.0.0 255.255.255.0 – określenie podsieci dla puli DHCP
  • default-router 192.168.0.1 – wskazanie bramy domyślnej przekazywanej klientom DHCP
